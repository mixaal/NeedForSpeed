# Need for Speed

The old 1995 Need for Speed  remake and _work in progress_ - this time in Unity 3d:


## Quickstart

```
git clone git@github.com:mixaal/NeedForSpeed.git
cd NeedForSpeed/
git lfs fetch
git lfs checkout

```
